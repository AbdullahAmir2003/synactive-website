## 🌟 Features

- **Modern Dark-Theme UI:** A custom-built, responsive dark theme using CSS Flexbox and Grid to ensure perfect rendering across desktop, tablet, and mobile devices.
- **Custom Iconography:** Lightweight, inline SVG icons used for all service pillars to ensure crisp scaling and fast load times.
- **Optimized Lead Capture:** Service-specific contact forms for security and compliance inquiries (Penetration Testing, ISO Compliance, AI Security Assessments).
- **Performance First:** Built as a lightweight, responsive static website to guarantee lightning-fast page loads and optimal SEO performance without the bloat of heavy JavaScript frameworks.

---

## 💻 Tech Stack

- **HTML5:** Semantic markup for accessibility and SEO.
- **CSS3:** Custom stylesheet (`assets/style.css`) utilizing modern CSS Grid, Flexbox, and CSS variables for brand consistency (e.g., Synactive Teal: `#0cbcb9`).
- **Formspree:** Serverless form handling for all contact and lead generation inquiries.

## 🛠 Local Development

Because this is a static HTML/CSS site, no complex build tools (such as Webpack, Vite, or Node.js) are required.

### Running the Site Locally

1. Clone the repository.
2. Open the project folder in your preferred editor (e.g., VS Code).
3. Install and run the **Live Server** extension.
4. Launch `index.html`.
5. Open the generated local URL (typically `http://127.0.0.1:5500`).
6. Edit HTML/CSS files and refresh automatically through Live Server.
